# Your DocuMind AI

##  🧠 AI Code Companion & Research Assistant

## Description
DocuMind AI is an intelligent bot that assists you based on the document you provide. This is done using streamlit, RAG(Retrieval Augmented Generation) with the help of Ollama Embedding model and DeepSeek-R1. Langchain and its components like Prompt templates play a major role in helping build this DocuMindAI. 

## Installation
1. Clone the repository
2. Navigate to the project directory
3. streamlit, langchain_core, langchain_community, langchain_ollama, pdfplumber

## Usage
- Run the AI Code companion 
  streamlit run app.py
- Run the Research Assistant
  streamlit run rag_deep.py

## Technologies Used
- Streamlit
- LangChain
- Ollama
- pdfplumber (For document processing)

## Features
You make your routine and provide me the document, you forget and I always guide you as a friendly assistant even when you forget important things that you mention.... As a research assistant, trust me bro nothing outside the information you provide.... Only based on the provided document.
